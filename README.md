# Support Team Schedule

Ứng dụng web lập lịch trực tuần 24/7 cho đội CSKH, xây dựng bằng React + TypeScript + Tailwind + dnd-kit + Zustand.

## Chạy nhanh

```bash
npm install
npm run dev -- --host 0.0.0.0 --port 5173
```

Mở: `http://localhost:5173`
